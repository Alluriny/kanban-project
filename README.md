# Kanban Board

Веб-приложение для управления задачами в формате Kanban-досок. Реализована авторизация, работа с досками, колонками и карточками, включая перетаскивание элементов.

## Функциональность

- Регистрация и авторизация пользователей (JWT)
- Создание, редактирование и удаление досок
- Создание, редактирование и удаление колонок
- Создание, редактирование и удаление карточек
- Перетаскивание карточек между колонками (Drag-and-Drop)
- Назначение исполнителей и установка дедлайнов
- Комментарии к карточкам
- Тёмная тема оформления
- Мок-режим для работы без бэкенда

## Запуск проекта

### Требования

- Node.js 16+
- Python 3.11+ (для бэкенда)
- PostgreSQL (опционально)

### Запуск фронтенда

```bash
cd kanban-frontend
npm install
npm start

Приложение будет доступно по адресу: http://localhost:3000

Запуск бэкенда 
bash
cd kanban-backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python init_db.py
uvicorn app.main:app --reload --port 8000
Swagger документация: http://localhost:8000/docs

Настройка режима работы
В файле kanban-frontend/src/api/api.js:

javascript
const USE_MOCK = true;   // Мок-режим (без бэкенда)
const USE_MOCK = false;  // Реальный бэкенд
